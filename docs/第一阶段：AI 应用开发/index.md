# 第一阶段：AI 应用开发

探究技术的本质，享受技术的乐趣！你好，我是长歌，欢迎来到 LLM Learning 中第一阶段的开篇！

在这一阶段的核心目标：**将 LLM 的能力落地到业务中，掌握 AI 应用开发。**

在这个阶段中，我会先给你介绍 LLM 的两大能力以及大模型的本质，理解了大模型的本质之后，我们就会以一个项目为导向，去学习开发不同类型的 AI 应用，在开发 AI 应用的过程中，我们最开始会使用Open AI SDK 开发一遍，让我们知道我们应该如何与大模型交互。当我们使用 OpenAI SDK 开发一遍之后，我们会利用 LangChain 改造一遍。到此整个阶段的核心就结束了，到了这个时候，你就能自己开发自己的 AI 应用了。以下是暂时规划出来的文章列表：

## 本阶段计划

- LLM 的本质
- 提示词工程
- 认识 OpenAI API
- ChatBot 类 AI 应用
- RAG 类 AI 应用
- Agent 类 AI 应用

## 建议阅读方式

1. 先建立整体认知。
2. 每学一个主题，就配合项目实践一起动手。
3. 不追求一开始就做复杂系统，先把最小可运行版本做出来。
