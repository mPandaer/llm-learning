# 路线图

这个教程分为两个阶段推进。

## 第一阶段：AI 应用开发

目标：先建立“能把模型真正用起来”的工程能力。

计划覆盖：

- 环境准备
- LLM 基础认知
- Prompt Engineering
- Embedding 与检索
- RAG
- Agent
- 工作流与项目实践

## 第二阶段：大模型原理与算法

目标：从知道怎么用，过渡到理解为什么它这样工作。

计划覆盖：

- Tokenizer
- Transformer
- 预训练
- 指令微调
- 参数高效微调
- 推理与部署
- 算法复现

## 更新原则

- 文档优先服务于理解
- 代码优先服务于验证
- 每新增一块知识，尽量补一份最小可运行示例
