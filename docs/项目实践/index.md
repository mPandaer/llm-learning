# 项目实践

这里用于索引教程配套的代码示例与实验项目。

## 组织方式

代码统一放在仓库根目录的 `projects/` 中。

建议后续按阶段组织：

- `projects/phase-1/`：AI 应用开发相关项目
- `projects/phase-2/`：大模型原理与算法相关项目

## 后续目标

每篇核心教程尽量配套一个最小可运行项目，例如：

- Prompt playground
- RAG demo
- Agent demo
- Tokenizer demo
- Transformer demo
- Finetune demo
